## Naming Convention

All traits follow the pattern:
```
{SS_tier}_{archetype}_{name}
```

Examples:
- `SS3_milady_Bera Headband` → [view trait](accessories/hats/README.md#bera-headband)
- `SS4_freetekno_MF Doom`
- `SS2_acidhouse_Mr Acid House`

Traits without SS prefix are common/standard rarity.
