# Elements

*The four classical elements that govern Mibera's cosmology.*

---

## Overview

Each Mibera is assigned an element based on their birth coordinates. The globe is divided into four quadrants, each mapped to a classical element:

| Element | Quadrant | Latitude | Longitude |
|---------|----------|----------|-----------|
| Air | Northeast | ≥ 0 (North) | ≥ 0 (East) |
| Earth | Northwest | ≥ 0 (North) | < 0 (West) |
| Fire | Southeast | < 0 (South) | ≥ 0 (East) |
| Water | Southwest | < 0 (South) | < 0 (West) |

---

## Air

**Name:** Air

![Air](https://mibera.fsn1.your-objectstorage.com/components/Elements/Air.webp)

**Mibera Geography Associations:** Northeast quadrant — Europe, Middle East, Asia, parts of Africa

**Real World Astrological Associations:** Gemini, Libra, Aquarius

**Quadrant:** Northern (lat ≥ 0) and Eastern (lon ≥ 0)

**Qualities:** Intellect, communication, movement, ideas, breath

## Earth

**Name:** Earth

![Earth](https://mibera.fsn1.your-objectstorage.com/components/Elements/Earth.webp)

**Real World Astrological Associations:** Taurus, Virgo, Capricorn

**Quadrant:** Northern (lat ≥ 0) and Western (lon < 0)
